What is CSS?
  1. CSS stands for cascading style sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.
  3. CSS is not neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  4. External stylesheet are stored in CSS files.
CSS can be added to HTML document in 3 ways:
  • inline - by using the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files.However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate this is easier to demonstrate, and easier for you to try it yourself.

When compariing CSS class vs ID, the difference is that the CSS class applies a style to multiple elements.ID,on the other hand ,applies a style to one unique element.IDis also special in that you can use a special URL to link to an element and it's used by JavaScript.

In CSS, slectors are used to target a specific elemnt or range of elements of a web page.Once an element has been targeted , a style or a set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
